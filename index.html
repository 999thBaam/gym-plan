Progression

Hit the top of the rep range on every set → add 2.5–5 kg next time and drop back to the bottom. Log every set.

Machine busy?

Every lift lists all the ways to do it, with photos. Any variant hits the same muscle — do it with the same sets × reps. Never wait in a queue.

Recovery valve (big calorie deficit)

If strength drops 2 weeks straight or sleep < 6 h: cut every 4-set lift to 3 sets and skip the last exercise. Every 6th week = half volume. Sharp pain → stop that lift; soreness is fine.

Daily frame

Warm-up: 5 min brisk walk + 2 light ramp sets on the first lift. Finish with a 10–15 min incline walk. Protein around the workout as planned.

Sunday · Rest

Full rest from lifting. Shopping + meal-prep day. Keep the 8–10k steps.

Optional: 20–30 min easy walk. Nothing harder — six lifting days only work if this one stays empty.
